== Description ==

Install a calculator on any page or post which helps to work out the cost savings of installing a gps tracking system in your fleet of vehicles.

= Features =

* Runs on JS
* Lightweight
* Calculator split onto multiple pages for ease of viewing
* Inherits the style of your theme.
* Shortcode available

== Changelog ==

= 1.0 =
* Plugin created.

== Installation ==

Install the plugin, activate it and then use the shortcode on any post or page.

[gps_roi_calculator]

== Frequently Asked Questions ==

= Why did you do this? =

We needed a simple calculator to assist our potential customers to work out the increased revenue they would see by installing a gps tracking system. 

= How do I edit the values? =

The non user input values which are used in the calculation can only be edited through code changes. 

= How are the emails sent? =

It will use whatever SMTP plugin you have installed.
